# Aurelian Website

Aurelian is a luxury jewelry homepage built with HTML and CSS. The design focuses on timeless gold accessories, elegant product sections, brand storytelling, and a refined editorial shopping experience.

## Features

- Hero section with luxury jewelry branding
- Product collection cards
- Brand story section
- Best seller product showcase
- New arrivals and craftsmanship blocks
- Lookbook image gallery
- Email subscription section
- Footer with shop, collection, brand, and customer care links
