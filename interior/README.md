# Interior Design Website UI

A modern, clean interior design landing page UI built using pure HTML and CSS. This project focuses on layout, spacing, and visual hierarchy inspired by real-world design websites.

## Features

- Hero section for the interior design brand
- Services section
- Story-driven about section
- Featured projects layout
- Image-led design presentation
