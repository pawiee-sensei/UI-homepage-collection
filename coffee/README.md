# Velora Brew Website

Velora Brew is a cafe and dining homepage UI built with HTML and CSS. The page highlights crafted food, warm cafe imagery, and restaurant-style sections for storytelling and menu presentation.

## Features

- Hero section for the cafe brand
- Food and drink visual sections
- Story-focused content block
- Dish showcase
- Space and atmosphere gallery
