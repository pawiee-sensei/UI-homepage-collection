# Mizu Matcha Website

Mizu is a modern, minimalist e-commerce website concept focused on premium Japanese matcha. The layout uses clean spacing, structured sections, and a calm green visual style to support the wellness-focused brand direction.

## Features

- Hero section for the matcha brand message
- Product collection showcase
- Matcha benefits section
- Preparation tutorial layout
- Journal-style content cards
- Email signup call-to-action
