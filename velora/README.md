# Velora Website

Velora is a lifestyle apparel homepage UI built with HTML and CSS. The page presents everyday essentials, community imagery, and product sections in a clean editorial style.

## Features

- Fashion homepage hero section
- Everyday essentials product blocks
- Varsity essentials section
- Community image gallery
- Lifestyle-focused layout
